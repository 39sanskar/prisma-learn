# learning prisma-orm

## Why use ORM:

- Easier to write and read: You use the same language you're coding in (like JavaScript).

- Safer: It helps prevent mistakes like SQL injection.

- Faster development: You don't have to write repetitive SQL for common actions (like creating, reading, updating data).

- Clear models: You define your data as code, so it's easy to see the structure of your database.
